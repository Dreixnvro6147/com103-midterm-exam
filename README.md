# com103-midterm-exam

What it does: This is a simple MLBB tool to help you track your recent games. Just enter your name and rank, then input your scores for 4 matches. The code does the math for you, showing your KDA and win rate, while also pointing out which of those matches was your best one.

How to run it: If you are using a phone, just use Pydroid 3, paste the code, and hit run. If you are on a computer, save your work as mlbb_tracker.py and use the terminal to launch it with Python.
